CLASS
CONSUMER

WHY NOW
Live audit: the level ladder duplicated its text labels — the drawing added no
value over the words because it showed no position or proximity. It didn't show
where price sits in the structure or how far each level is.

MAX EXPECTED DELTA
40 production LOC (dashboard_renderer.py _render_level_diagram + caller).

GOAL
Make the ladder show what the text can't: a NOW marker at current price, a signed
% distance from NOW on every level label, and a faint band over the NOW→ENTRY gap.

SCOPE
- Pass the candidate's current_price into _render_level_diagram as `now_price`.
- Render a distinct NOW marker (white line + dot + "NOW <price>" label; it is the
  0% reference, no % suffix) and include it in the y-scale.
- Append a signed % distance from NOW to every fib / zone / VWAP / ENTRY label.
- Shade the gap between NOW and the ENTRY trigger with a faint band.

OUT OF SCOPE
- A true ENTRY→INVALIDATION risk band: `invalidation` is prose ("reclaims
  PRIOR_LOW …"), not a numeric level, so there is no price to shade. The band
  here is NOW→ENTRY (distance-to-trigger). A numeric invalidation/stop would be an
  upstream contract change, not presentation — deferred.
- A real intraday price series / candlesticks (needs bar data; separate PRD).

FILES
M cuttingboard/delivery/dashboard_renderer.py
M tests/test_dash_level_diagram.py

CHANGE SURFACE
cuttingboard/delivery/dashboard_renderer.py — _render_level_diagram (new optional
`now_price`, band rect, NOW marker, % suffix on labels) and its single caller in
_render_candidate_card. The ENTRY line y stays a pinned contract; NOW uses a
distinct colour so it never collides with the ENTRY-line test regex.

REQUIREMENTS

R1 — NOW marker
When current price is available, a distinct NOW marker renders at its y with a
"NOW <price>" label.
FAIL: with current_price present, no NOW marker/label renders.

R2 — % distance on labels
Every non-NOW level label carries a signed % distance from NOW.
FAIL: a fib/zone/VWAP/ENTRY label lacks its % suffix when NOW is present.

R3 — Entry-gap band
A faint band shades between NOW and ENTRY when they differ.
FAIL: NOW and ENTRY differ but no band renders.

R4 — Graceful without NOW
With no current_price the diagram renders as before (no marker, no %, no band).
FAIL: absent current_price produces a NOW marker, a % suffix, or a crash.

DATA FLOW
entry.current_price → now_price → NOW marker + %-distance suffixes + NOW→ENTRY band

FAIL CONDITIONS
- current_price present but no NOW marker.
- Missing % suffix on a level label when NOW present.
- NOW≠ENTRY but no band.
- current_price absent yet a marker/%/band appears.

VALIDATION
Manual:
- Render a card with current_price: NOW marker sits at price; labels read
  "<name> <price> ±x.x%"; a faint band spans NOW→ENTRY.
- Render without current_price: unchanged from PRD-216.
Automated:

  tolerate the % suffix.
