3. **REQUIRED — V11 does not prove that the supplied object is a

   `<sha>^{commit}`.

   from current `HEAD` (`SKILL.md:161`, V12 at line 202). If the
   reviewed branch tip is not checked out, the two fields would describe
   different histories while both checks pass. Compute both fields from
   the same reviewed commit.
