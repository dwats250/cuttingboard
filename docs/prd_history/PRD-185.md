PRD-185 — Bump GitHub Actions to Node 24 majors (checkout v6, setup-python v6, upload-artifact v7)

WHY NOW
`actions/checkout@v4`, `actions/setup-python@v5`, and
`actions/upload-artifact@v4` run on Node 20, which GitHub forces to Node 24
on 2026-06-02 and removes from runners on 2026-09-16. Every CI run (and
every live-pipeline run) currently logs the deprecation warning observed on
the PRD-184 PR checks.

MAX EXPECTED DELTA
12 changed `uses:` lines across 5 workflow files. No logic, schedule,
step, or input change.

GOAL
Eliminate the Node 20 deprecation warning by pinning the three actions to
their current Node-24 majors across all workflows, with no behavior change.

SCOPE
- `actions/checkout@v4` -> `@v6` (5 workflows).
- `actions/setup-python@v5` -> `@v6` (4 workflows).
- `actions/upload-artifact@v4` -> `@v7` (3 workflows).

OUT OF SCOPE
- No change to any step, `with:` inputs, schedule, trigger, run command,
  or gating logic. Version-pin bumps only.
- No new actions; no change to non-action workflow content.
