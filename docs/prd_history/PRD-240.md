REQUIREMENTS

R1 — Remove the EXPANSION R:R discount
`EXPANSION_RR_RATIO = 2.0`. Audit's strongest finding: momentum/breakout literature
(SSRN 4729284, 6592020; search-snippet confidence) pairs continuation setups with lower
win rates needing HIGHER R:R, not lower; 2.0 removes the unsupported discount without
claiming calibration for more (no in-repo win-rate data exists). Side effect:
continuation max stop width tightens 2.0×→1.5×ATR14 (interacts with R6).
FAIL: an EXPANSION-regime candidate with target-based R:R 1.8 passes Gate 7.

R2 — Gate 6 ATR stop floor 1.0×
Gate 6's inline `0.5` ATR multiplier becomes config `STOP_ATR_FLOOR_K = 1.0` (DIRECT path only; every convention the audit found starts ≥ 1×). 1% floor unchanged.
FAIL: a candidate with stop ≥ 1% of price but risk 0.75×ATR14 passes Gate 6.

R3 — Named constants replace disguised literals (behavior-neutral)
`MIN_STOP_PCT = 0.01` shared by the Gate 6 `stop_pct` check and the continuation
`min_risk` check (two independent literals today); `CONTINUATION_REWARD_ATR_MULTIPLE = 3.0`
replaces the continuation reward expression — arithmetically a constant 3.0×ATR14 dressed
as breakout-window arithmetic — commented as a fixed ATR multiple / stop-width ceiling.
FAIL: `rg "0\.01" cuttingboard/qualification.py` matches a stop-floor literal, or the reward expression still references the breakout-window constants.

R4 — Regime-tiered R:R in `_resolve_entry_mode`
The FVG pullback R:R minimum comes from the same three-tier regime logic as Gate 7
(one shared helper), adding the missing EXPANSION branch — behavior-neutral once R1
lands; prevents silent divergence on future retunes.
FAIL: with `EXPANSION_RR_RATIO` monkeypatched ≠ `MIN_RR_RATIO`, `_resolve_entry_mode` applies `MIN_RR_RATIO` to an EXPANSION-regime candidate.

R5 — Continuation momentum conviction
INSUFFICIENT_MOMENTUM also fires unless `(Close − Low) / (High − Low) ≥ 0.75` (path is
LONG-only by construction; reuses the FVG displacement quartile) — a wick-dominated
candle is not momentum even when its range clears 0.75×ATR14.
FAIL: a breakout candle with range ≥ 0.75×ATR14 but close_location < 0.75 qualifies.

R6 — Continuation stop-floor asymmetry retained, documented
No ATR floor on the continuation stop check; a code comment there records why: the
stop anchors to the structural breakout level (chosen-stop convention does not map),
and a floor would combine with the post-R1 ceiling (risk ≤ 1.5×ATR14) into a
0.5×ATR-wide qualifying band — a de facto path shutdown nobody has decided on.
FAIL: the continuation stop block gains an ATR floor, or carries no such comment.
