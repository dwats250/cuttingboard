# PRD-092 - Macro Conditions Consolidation

**Status:** DRAFT
**Owner:** Codex

---

## GOAL

Reduce top-level block count by moving Macro Pressure inside Macro Tape as a collapsed `<details>` sub-section. Raw macro driver values, tradable ticker prices, and all pressure interpretation remain fully accessible; pressure is collapsed by default.

---

## SCOPE

1. Modify `cuttingboard/delivery/dashboard_renderer.py`.
2. Modify `tests/test_dashboard_renderer.py`.
3. The standalone `<div class="block" id="macro-pressure">` block is removed.
4. `<details id="macro-pressure">` with `<summary>Macro Pressure</summary>` is appended inside the `id="macro-tape"` block, before the tape block's closing `</div>`.
5. The `<h2>Macro Pressure</h2>` line is removed; it is replaced by the `<summary>` element.
6. All existing Macro Pressure content moves inside the `<details>` wrapper unchanged: pressure-grid with component badges, overall badge, NO PRESSURE DATA guard, FIELD_MISSING guard.
7. CSS additions:
   - Extend the existing `summary` cursor/list-style selector to include `#macro-pressure summary`.
   - Extend the existing `webkit-details-marker` selector to include `#macro-pressure summary::-webkit-details-marker`.
   - Add a `#macro-pressure summary` rule: `color:#aaa;font-size:0.7rem;text-transform:uppercase;letter-spacing:.05em` (consistent with `#run-history summary` style).
   - Add `#macro-pressure{margin-top:8px}` for spacing from the tradables grid.
8. `_macro_tape_block()` in `dash_helpers.py`, PA1, and PA2 do **not** require changes. They use `id="macro-pressure"` as an end-of-tape-content delimiter. This continues to work correctly because the bias/drivers/tradables grid content all precede the `<details id="macro-pressure">` element in both the old and new HTML structure.
9. `test_null_safe_secondary_sections_no_crash` does not require changes — `id="macro-pressure"` remains present on the `<details>` element.
10. Do not edit `ui/dashboard.html` or `ui/index.html`.

## IMPLEMENTATION NOTES

**Renderer change — tape block restructure:**

Before:
```python
    w('  </div>')           # closes macro-tradables-grid
    w("</div>")             # closes <div class="block" id="macro-tape">

    # --- macro-pressure ---
    w('<div class="block" id="macro-pressure">')
    w("  <h2>Macro Pressure</h2>")
    if (not macro_drivers) or ...:
        w('  <div class="pressure-no-data">NO PRESSURE DATA</div>')
    elif not isinstance(pressure, dict):
        w('  <div class="pressure-no-data">FIELD_MISSING</div>')
    else:
        w('  <div class="pressure-grid">')
        ...
        w('  </div>')
    w("</div>")
```

After:
```python
    w('  </div>')           # closes macro-tradables-grid
    w('  <details id="macro-pressure">')
    w('    <summary>Macro Pressure</summary>')
    if (not macro_drivers) or ...:
        w('  <div class="pressure-no-data">NO PRESSURE DATA</div>')
    elif not isinstance(pressure, dict):
        w('  <div class="pressure-no-data">FIELD_MISSING</div>')
    else:
        w('  <div class="pressure-grid">')
        ...
        w('  </div>')
    w('  </details>')
    w("</div>")             # closes <div class="block" id="macro-tape">
```

**CSS additions** — extend existing selectors (do not duplicate):
- Add `#macro-pressure summary` to the summary `cursor:pointer;list-style:none` selector.
- Add `#macro-pressure summary::-webkit-details-marker` to the webkit marker `display:none` selector.
- Add new rule: `#macro-pressure summary{color:#aaa;font-size:0.7rem;text-transform:uppercase;letter-spacing:.05em}`
- Add new rule: `#macro-pressure{margin-top:8px}`

**Why PA1/PA2/`_macro_tape_block()` need no changes:**
These helpers extract tape content by splitting on `id="macro-pressure"` as an end delimiter. After the change, `id="macro-pressure"` appears after the bias/drivers/tradables grid content — the same relative position as before. The split continues to correctly isolate the tape-only content.

Before: Two sibling top-level blocks. Macro Pressure always fully visible.
After: One tape block containing drivers, grid, and a collapsed pressure section. Page is shorter by one block in default view.
