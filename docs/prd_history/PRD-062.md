**Status:** COMPLETE
**Commit:** —

---

## GOAL

Add a deterministic Macro Tape value row to the dashboard renderer without
changing Macro Tape arrow behavior, macro bias behavior, no-data behavior, or
any upstream schema/runtime surface.

---

## SCOPE

- Update only the existing Macro Tape rendering path in
  `cuttingboard/delivery/dashboard_renderer.py`
- Add targeted renderer tests in `tests/test_dashboard_renderer.py`
- Preserve renderer-only boundaries

---

## OUT OF SCOPE

- Any runtime, payload, contract, market map generation, notification, or trade
  logic change
- Any upstream addition of `current_price`
- Any dashboard change outside the Macro Tape block

---

## FILES

M `cuttingboard/delivery/dashboard_renderer.py`
M `tests/test_dashboard_renderer.py`
M `docs/PRD_REGISTRY.md`

---

## NOTES

- Row 1 remains the existing ticker/direction row.
- Row 2 uses `class="macro-tape-values"` and fixed symbol-associated slots.
- ETF values render `--` unless `market_map["symbols"][symbol]["current_price"]`
  already exists and is a finite non-boolean number.
