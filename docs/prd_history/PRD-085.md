# PRD-085 — Regression Coverage: current_price Survives Full Runtime Processing Chain

**Status:** IN PROGRESS

## GOAL

Add regression tests proving `current_price` produced by `build_market_map()` is preserved through `inject_lifecycle()` and is present in the write-ready dict passed to `_write_market_map_file()`, closing future regression risk on the full in-memory market_map write path.

## CONTEXT

PRD-084 added `"current_price": price` to `_build_symbol_record()`.

Direct testing confirms `build_market_map()` includes `current_price` for `PRIMARY_SYMBOLS`.

Code inspection confirms no known function between `build_market_map()` and `_write_market_map_file()` strips `market_map["symbols"]` fields:
- `build_visibility_map()` builds a separate result dict and does not mutate `market_map`
- `apply_overnight_policy()` deepcopies contract and reads `market_map` read-only
- `inject_lifecycle()` deepcopies `current_map` and adds lifecycle keys only
- `_write_market_map_file()` serializes the dict with `json.dumps`

The `logs/market_map.json` artifact on disk may still lack `current_price` because no confirmed successful full pipeline run has completed since PRD-084 merged. This PRD adds the missing regression coverage to guarantee durability of the PRD-084 fix.

## REQUIREMENTS

**R1 — inject_lifecycle preserves current_price**
`inject_lifecycle()` must not drop `current_price` from any symbol record.
FAIL: `inject_lifecycle(build_market_map(...), previous_map=None)` output lacks `current_price` on any configured `PRIMARY_SYMBOL`.

**R2 — Full in-memory write chain preserves current_price**
The dict produced by `inject_lifecycle(build_market_map(...), previous_map=None)` must include `current_price` for all `PRIMARY_SYMBOLS` before serialization. Values may be numeric or None; the key must be present.
FAIL: `current_price` is absent from any symbol record in the write-ready dict.

**R3 — No unnecessary production changes**
R1 and R2 satisfied by tests only unless a test reveals an actual production defect.
FAIL: production source files modified without a failing test demonstrating the need.

## TESTS

**T1** — `inject_lifecycle` preserves `current_price` (`tests/test_market_map_lifecycle.py`)
Build a `market_map` with valid normalized quotes for all `PRIMARY_SYMBOLS`, pass through `inject_lifecycle(..., previous_map=None)`, assert `current_price` is present and equals source quote price for all `PRIMARY_SYMBOLS`.

**T2** — Full in-memory chain: build → lifecycle → write-ready dict (`tests/test_market_map.py`)
Same build, same lifecycle pass, assert all six symbols have `current_price` present in the output dict.
