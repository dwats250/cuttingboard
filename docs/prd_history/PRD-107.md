# PRD-107 ‚Äî Trend Structure Snapshot Sidecar

STATUS
IN PROGRESS

GOAL
Add a deterministic Trend Structure Snapshot sidecar that records
higher-timeframe and intraday structure context for the configured
tradable symbols. The sidecar must improve trade evaluation by exposing
whether a symbol is structurally aligned, extended, conflicted, or
unsuitable for new entries.

This PRD is sidecar-only. It must not mutate trade decisions, qualification
logic, runtime outcome, contract outcome, alerts, dashboard HTML, payload,
or existing market_map grading.

SCOPE
- Add a new pure builder module: `cuttingboard/trend_structure.py`.
- Build one deterministic record per configured tradable symbol.
- Persist output to `logs/trend_structure_snapshot.json` via a runtime helper.
- Use only inputs already produced by runtime upstream of qualification:
  normalized quotes (`cuttingboard/normalization.py NormalizedQuote`) and the
  per-symbol OHLCV DataFrames already fetched for `cuttingboard/derived.py`.
- Represent missing data with explicit `null` and `UNKNOWN` values.
- Add unit tests for deterministic output, missing data, and artifact shape.
- Preserve all existing artifacts and contracts.

OUT OF SCOPE
- Trade decision, qualification, market_map grade, dashboard, notification,
  payload, contract, alerting, broker, backtest, ML, options chain mutation.
- New symbols. Network calls. Direct file I/O inside the builder.
- Hard-blocking entries on trend_structure values.

INPUTS (binding)
- `normalized_quotes: dict[str, NormalizedQuote]` ‚Äî
  `current_price = normalized_quotes[symbol].price`.
- `history_by_symbol: dict[str, pandas.DataFrame]` of OHLCV bars keyed by
  symbol with columns `Open, High, Low, Close, Volume`. Populated in runtime
  by reusing the existing `fetch_ohlcv()` results gathered for
  `cuttingboard/derived.py`. Runtime should pass through whatever OHLCV
  history is already available. If fewer than 50 or 200 bars are available,
  the builder must return null for unavailable SMA fields. Expanding fetch
  depth to 200 bars is allowed only if the existing fetch helper already
  supports that parameter without changing callers outside this PRD. The
  builder MUST NOT extend, fetch, or coerce missing series.
- `symbols: list[str]` =
  `[s for s in config.ALL_SYMBOLS if s not in config.NON_TRADABLE_SYMBOLS]`.
- `generated_at: datetime | None` ‚Äî timezone-aware. Builder MUST NOT call
  `datetime.now()`. Naive datetime raises `ValueError`.

DATA CONTRACT
```
{
  "schema_version": 1,
  "generated_at": "<iso8601 utc with tz, or null>",
  "source": "trend_structure",
  "symbols": {
    "<SYMBOL>": {
      "symbol": "<SYMBOL>",
      "data_status": "OK | PARTIAL | MISSING",
      "current_price": number | null,
      "vwap": number | null,
      "sma_50": number | null,
      "sma_200": number | null,
      "relative_volume": number | null,
      "price_vs_vwap": "ABOVE | BELOW | UNKNOWN",
      "price_vs_sma_50": "ABOVE | BELOW | UNKNOWN",
      "price_vs_sma_200": "ABOVE | BELOW | UNKNOWN",
      "trend_alignment": "BULLISH | BEARISH | MIXED | UNKNOWN",
      "entry_context": "SUPPORTIVE | AVOID | NEUTRAL | UNKNOWN",
      "reason": "<short deterministic reason>"
    }
  }
}
```

FIELD RULES
- `schema_version`: integer literal `1`.
- `generated_at`: ISO-8601 UTC string with tzinfo when provided; else `null`.
- `symbols`: keys are exactly the `symbols` argument (set equality).
- `current_price`: `normalized_quotes[symbol].price`; `null` if symbol absent.
- `sma_50`: mean of last 50 daily closes when ‚â•50 bars; else `null`.
- `sma_200`: mean of last 200 daily closes when ‚â•200 bars; else `null`.
- `vwap`: cumulative `sum((H+L+C)/3 * V) / sum(V)` over the most recent
  intraday session bars in `history_by_symbol[symbol]`. If the DataFrame
  contains only daily bars, `vwap = null`.
- `relative_volume`: `today_volume / mean(volume over previous 20 daily bars)`.
  `null` when fewer than 21 bars or `today_volume` missing.
- `price_vs_<ref>`: `"ABOVE"` if `current_price > ref`; `"BELOW"` if `<`;
  `"UNKNOWN"` if either side null OR exact equality. There is no `"AT"`.
- `data_status`:
  - `MISSING` = `current_price is None`.
  - `OK` = `current_price` and all three of `{vwap, sma_50, sma_200}` are
    not null.
  - `PARTIAL` = otherwise (current_price present, ‚â•1 reference null).
- `trend_alignment`:
  - `BULLISH` = `price_vs_sma_50 == ABOVE AND price_vs_sma_200 == ABOVE`.
  - `BEARISH` = `price_vs_sma_50 == BELOW AND price_vs_sma_200 == BELOW`.
  - `MIXED`   = one ABOVE and one BELOW.
  - `UNKNOWN` = either is UNKNOWN.
- `entry_context`:
  - `SUPPORTIVE` = `BULLISH AND price_vs_vwap == ABOVE`.
  - `AVOID`      = `BEARISH AND price_vs_vwap == BELOW`.
  - `UNKNOWN`    = `data_status == MISSING OR alignment == UNKNOWN OR
                    price_vs_vwap == UNKNOWN`.
  - `NEUTRAL`    = otherwise.
- `reason`: deterministic short string composed only from this record's
  computed fields. No timestamps, no random text.

RUNTIME ORDERING
- Build snapshot AFTER `normalized_quotes` are produced and AFTER OHLCV
  history is gathered for `derived.py`, AFTER `_write_market_map_file()`,
  BEFORE returning success.
- Persist via `_write_trend_structure_snapshot()` using the existing atomic
  `.tmp ‚Üí replace` pattern.
- Snapshot is NOT read by qualification, contract, payload, notifications,
  or dashboard.
- Snapshot is NOT added to `latest_hourly_contract.json` or
  `latest_hourly_payload.json`.

FAIL CONDITIONS
- R1 ‚Äî Any change to trade decision, qualification, market_map grading,
  dashboard, notification, contract, or payload outputs.
  FAIL: diff on those artifacts/files outside the PRD-listed FILES.
- R2 ‚Äî `trend_structure_snapshot` referenced in `contract.py`, `delivery/`,
  `notifications/`, or `ui/` source.
  FAIL: leakage grep returns matches.
- R3 ‚Äî Builder raises on missing data instead of producing null/UNKNOWN.
  FAIL: missing-quote/history fixtures raise.
- R4 ‚Äî Tests pass only because fixtures silently omit symbols.
  FAIL: snapshot symbol set ‚âÝ requested `symbols` argument.
- R5 ‚Äî Artifact contains any symbol in `config.NON_TRADABLE_SYMBOLS`.
  FAIL: validation script finds `^VIX`, `^TNX`, `DX-Y.NYB`, or `BTC-USD`
  in `data["symbols"]`.
- R6 ‚Äî Builder performs network calls or file I/O.
  FAIL: import-time or call-time I/O detected.
- R7 ‚Äî Builder calls `datetime.now()`.
  FAIL: source contains `datetime.now`.
- R8 ‚Äî `sma_50`/`sma_200`/`vwap` silently computed from insufficient bars.
  FAIL: short-history fixture yields non-null SMA.
- R9 ‚Äî `"AT"` appears as a value in any `price_vs_*` field.
  FAIL: equality fixture yields `"AT"`.
