# PRD-037 — Dashboard Publish Artifact

**Status:** IN PROGRESS
**Author:** Dustin Watson
**Date:** 2026-04-28

---

## GOAL

Make the read-only dashboard generated by PRD-036 viewable from a phone through a static GitHub Pages-compatible artifact.

---

## SCOPE

- Copy `reports/output/dashboard.html` to `docs/dashboard.html`
- Preserve dashboard content exactly
- Add validation that the published file matches the generated file
- Keep publishing separate from renderer logic

---

## OUT OF SCOPE

- Dashboard rendering changes
- New UI sections
- New trading logic
- Live data fetching
- GitHub Pages configuration changes
- Deployment automation beyond file copy/validation

---

## FILES

ADD:
- `tools/publish_dashboard.py`
- `tests/test_publish_dashboard.py`

DO NOT MODIFY:
- `cuttingboard/delivery/dashboard_renderer.py`
- `cuttingboard/delivery/payload.py`
- `cuttingboard/contract.py`
- `cuttingboard/runtime.py`

---

## REQUIREMENTS

**R1 — Source Artifact**
- MUST read `reports/output/dashboard.html`
- FAIL: `FAIL: source not found` if missing

**R2 — Publish Target**
- MUST write `docs/dashboard.html`
- MUST create `docs/` directory if missing

**R3 — Exact Copy**
- `docs/dashboard.html` MUST match `reports/output/dashboard.html` byte-for-byte
- FAIL: if SHA-256 of source ≠ SHA-256 of target

**R4 — Static Hosting Compatibility**
- Published file MUST be plain static HTML
- No JavaScript injection, no external assets, no redirect

**R5 — Separation of Concerns**
- `publish_dashboard.py` MUST NOT import engine, runtime, regime, qualification, market data, or renderer internals
- FAIL: if any forbidden module name appears in source

---

## DATA FLOW

```
reports/output/dashboard.html
        ↓

```

---

## FAIL CONDITIONS

- **F1** — `reports/output/dashboard.html` does not exist → exit 1
- **F2** — `docs/dashboard.html` SHA-256 ≠ source SHA-256 → exit 1
- **F3** — publisher modifies HTML content → test_exact_byte_copy fails
- **F4** — publisher imports forbidden modules → test_no_engine_imports fails
- **F5** — `docs/dashboard.html` not written → test_publishes_dashboard_html fails
