PIPELINE LAYERS BUILT

| Layer | Module | Role |
|-------|--------|------|
| 1 | config.py | All constants and secrets from .env |
| 2 | ingestion.py | RawQuote fetch — yfinance primary, Polygon fallback |
| 3 | normalization.py | NormalizedQuote — pct_change to decimal, UTC enforcement |
| 4 | validation.py | Hard validation gate — HALT_SYMBOL failure stops pipeline |
| 5 | derived.py | EMA9/21/50, ATR14 (Wilder RMA), momentum_5d, volume_ratio |
| 6 | structure.py | Per-ticker classification: TREND / PULLBACK / BREAKOUT / REVERSAL / CHOP |
| 7 | regime.py | 8-vote macro regime model → RISK_ON / RISK_OFF / NEUTRAL / CHAOTIC |
| 8 | qualification.py | Multi-gate trade qualification — hard and soft gates |
| 9 | options.py | Options expression engine — spread selection, DTE, strike distance |
| 10 | output.py + audit.py | Terminal render, ntfy alert, append-only JSONL audit |

REQUIREMENTS

R1 — Pipeline Completeness
All 10 layers MUST be implemented and wired in strict phase order.
No derived metric computed on unvalidated input.

R2 — Output Contract
Every run MUST produce exactly one of: TRADE | NO TRADE | HALT.
No ambiguous or partial outputs.

R3 — Halt on Core Symbol Failure
If any HALT_SYMBOL (^VIX, DX-Y.NYB, ^TNX, SPY, QQQ) fails validation,
the pipeline MUST stop immediately and output HALT.

R4 — Regime Engine
8-input vote model MUST classify macro regime as RISK_ON / RISK_OFF / NEUTRAL / CHAOTIC.
CHAOTIC override: VIX single-interval spike > 15% → CHAOTIC regardless of votes.

R5 — Qualification Gates
Hard gates (1–4): immediate REJECT. Soft gates (5–9): one miss → WATCHLIST, two+ → REJECT.

R6 — Ingestion Fallback
All symbols except ^VIX, DX-Y.NYB, ^TNX MUST attempt Polygon fallback on yfinance failure.

R7 — Test Coverage
All pipeline layers MUST have test coverage. 297 tests passing at delivery.

R8 — Scheduled Execution
GitHub Actions MUST run premarket (06:00 PT) and intraday (every 30 min, 14:00–21:00 UTC) jobs.
